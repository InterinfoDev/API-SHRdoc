# appFlowAgentDetail
查詢單筆簽核代理人詳細資料

### HTTP Request
```
https://114.34.125.246:8090/servlet/HRNative/appFlowAgentDetail
```

### HTTP Request Mehod
```
POST
```

### Request body
| Key | Value | Type | Description |
|:----------|:-------------|:-----|:------------|
| uid | 98599308101484732326 | String | 需透過appLogin取得
| right | 51341911904173543336756162544864820 | String | 需透過appLogin取得 |
| request | {'flowAgentKey':'4326084225293747024556419988109226030786170329214256400024434116838672769187526122578773346158145806659984556753490868565199631711318172'} | Object | 查詢條件(依據appFlowAgentList列表的隱藏欄位key值分解)

### fieldType
| Type | Description | key |
|:---------|:------------|:------------|
| text | 文字顯示類型  | fieldValue |

### Properties
| Property | Type | Description |
|:---------|:-----|:------------|
| uid   | String | 加密後帳號 |
| right | String | 加密後系統相關資料 |
| request | Object | 要求本文 |

```json
{
    "status": "fail",
    "code": 500,
    "message": [
        "查無資料"
    ],
    "data": {}
}
```
